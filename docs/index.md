# ИИ Бармен 🍸

Добро пожаловать в документацию проекта **ИИ Бармен** — интеллектуального бармен-консультанта на основе RAG (Retrieval-Augmented Generation) с микросервисной архитектурой.

## Быстрый старт

!!! tip "Начните здесь"
    Если вы хотите быстро запустить проект, перейдите в раздел [Быстрый старт](quickstart.md).

## Ключевые возможности

✅ **Интеллектуальные ответы** — использует YandexGPT для генерации рецептов  
✅ **Векторный поиск** — находит релевантные коктейли по описанию  
✅ **Модерация контента** — безопасное общение для пользователей 18+  
✅ **Микросервисная архитектура** — масштабируемая и отказоустойчивая  
✅ **Telegram интерфейс** — удобный доступ через популярный мессенджер
